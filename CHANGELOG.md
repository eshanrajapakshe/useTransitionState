# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added
- Initial release of `useTransitionState` hook
- Three built-in animation presets: `fade`, `slide`, and `zoom`
- Support for custom animation keyframes
- Lifecycle callbacks: `onEnter`, `onEntered`, `onExit`, `onExited`
- TypeScript support with full type definitions
- SSR compatibility with isomorphic layout effects
- Configurable duration and timing functions
- Comprehensive documentation and examples

### Features
- Drop-in replacement for `useState` with animation support
- Zero dependencies (React as peer dependency only)
- Tiny bundle size
- Works with any CSS properties
